# 融·金融计算器 (Rong Financial Calculator)

[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://pikuan.cn)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个功能全面、设计精美的金融计算器网页应用，支持多种贷款计算、反推利率、方案对比以及基础计算器工具。采用现代化UI设计，完美适配深色模式，并内置高精度计算引擎，满足个人贷款测算、方案对比及财务分析需求。

## ✨ 功能特性

### 💰 标准贷款计算
- 支持 **等额本息**、**等额本金**、**先息后本** 三种还款方式
- 利率输入支持 **年利率（%）** 或 **月利率（厘）**
- 期限支持 **年** 或 **月**，并提供快速期限按钮（1年/2年/…）
- 先息后本模式下可切换还款频率：**每月还款** 或 **每季度还款**
- 实时计算：**月供/季供**、**首月/季还款**、**总利息**、**还款总额**
- 生成 **详细还款计划表**（分页显示），并支持导出为 **Excel** 文件
- 提供 **本金-利息占比环形图**，直观展示利息比重

### 🔄 反推利率计算
- 已知每期还款额或总利息，反推 **年化利率**（等额本息模式）
- 已知每期利息，反推利率（先息后本模式）
- 同样支持还款频率切换，并生成对应的还款计划表与导出功能

### 📊 方案对比分析
- 可配置两组贷款方案（金额、利率、期限、还款方式）
- 对比 **月供**、**总利息**、**还款总额**，并用颜色标注差异
- 支持导出对比结果为 XLSX 文件

### 🧮 基础计算器工具
- 提供加、减、乘、除、百分比、退格、清空等基础运算
- 支持键盘输入（数字、运算符、回车、ESC等）

### 🌓 深色模式自适应
- 根据系统偏好自动切换深色/亮色主题
- 所有组件（卡片、按钮、输入框、表格）均适配深色样式

### 📱 响应式设计
- 移动优先，在小屏幕上自动调整布局和字号
- 表格支持横向滚动，分页按钮自适应

## 🛠️ 技术栈

- **HTML5** + **CSS3** + **JavaScript (ES6)**
- **Tailwind CSS** – 快速构建响应式界面
- **Chart.js** – 绘制本金-利息占比环形图
- **ExcelJS** – 生成并导出 XLSX 文件
- **decimal.js** – 高精度浮点数计算，避免金融计算误差
